# React Native - Desenvolva APPs Nativas para Android e iOS

![React Native](./assets/react-native-banner.webp)

Repositório com os projetos desenvolvidos durante o curso [React Native - Desenvolva APPs Nativas para Android e iOS](https://www.udemy.com/course/curso-react-native/), disponível na Udemy.

O curso aborda os fundamentos do React Native para criação de aplicativos móveis multiplataforma (Android e iOS) a partir de uma única base de código, cobrindo desde a estruturação de componentes até a construção de projetos práticos completos.

## 📱 Projetos

| Projeto | Descrição | Link |
| --- | --- | --- |
| Calculadora | Aplicativo de calculadora com operações matemáticas básicas | [Ver projeto](./calculadora) |
| Campo Minado | Aplicativo do clássico jogo Campo Minado | [Ver projeto](./campo-minado) |

Cada projeto possui seu próprio README com uma descrição detalhada e um vídeo de demonstração do funcionamento do aplicativo.

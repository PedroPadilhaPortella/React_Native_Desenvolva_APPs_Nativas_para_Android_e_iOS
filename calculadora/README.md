# 🧮 Calculadora

Aplicativo de calculadora desenvolvido em React Native, com suporte às operações matemáticas básicas (soma, subtração, multiplicação e divisão), como parte do curso [React Native - Desenvolva APPs Nativas para Android e iOS](https://www.udemy.com/course/curso-react-native/).

## 🎥 Demonstração

![Calculadora Demonstração](../assets/calculadora-demo.mp4)

## 🛠️ Tecnologias

- React Native
- TypeScript
- Expo

## ▶️ Como executar

```bash
cd calculadora
npm install
npx expo start
```

# 💣 Campo Minado

Aplicativo do clássico jogo Campo Minado (Minesweeper), desenvolvido em React Native como parte do curso [React Native - Desenvolva APPs Nativas para Android e iOS](https://www.udemy.com/course/curso-react-native/).

## 🎥 Demonstração

![Campo Minado Demonstração](../assets/campo-minado-demo.mp4)

## 🛠️ Tecnologias

- React Native
- TypeScript
- Expo

## ▶️ Como executar

```bash
cd campo-minado
npm install
npx expo start
```

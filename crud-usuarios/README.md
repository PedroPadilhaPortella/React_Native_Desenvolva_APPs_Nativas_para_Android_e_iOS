# CRUD de Usuários

Aplicativo simples de gerenciamento de usuários, desenvolvido em React Native como parte do curso [React Native - Desenvolva APPs Nativas para Android e iOS](https://www.udemy.com/course/curso-react-native/).

## 🎥 Demonstração

![CRUD de Usuários Demonstração](../assets/crud-usuarios-demo.mp4)

## 🛠️ Tecnologias

- React Native
- TypeScript
- Expo

## ▶️ Como executar

```bash
cd crud-usuarios
npm install
npx expo start
```
